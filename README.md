## üé® Intelligent Commit Messages

The enhanced script generates professional conventional commit messages:

### Conventional Commit Format
```
type(scope): description

Examples:
feat(auth): add user registration endpoint
fix(api): resolve null pointer in validation
docs(readme): update installation instructions
refactor(utils): simplify error handling logic
```

### Message Types & Emojis

| Type | Emoji | Purpose | Example |
|------|-------|---------|---------|
| `feat` | ‚ú® | New features | `feat(auth): add OAuth login` |
| `fix` | üêõ | Bug fixes | `fix(api): handle null responses` |
| `docs` | üìù | Documentation | `docs(readme): add usage examples` |
| `style` | üé® | Code formatting | `style(components): fix indentation` |
| `refactor` | ‚ôªÔ∏è | Code restructuring | `refactor(utils): extract helper functions` |

| `test` | ‚úÖ | Testing | `test(auth): add unit tests` |
| `build` | üîß | Build system | `build(deps): update dependencies` |
| `ci` | üë∑ | CI/CD | `ci(github): add workflow for testing` |
| `chore` | üîß | Maintenance | `chore(config): update eslint rules` |
